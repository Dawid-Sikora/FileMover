Aby program działał poprawnie należy importować go do środowiska programistycznego, które ma wsparcie dla architektury Maven (InteliJ, Eclipse).
Wymagana jest również posiadana Java minimum w wersji 11. 
Aplikację uruchamia się poprzez wywołanie metody main() w klasie PretiusApp.
Foldery na których działa aplikacja są zawarte w projekcie, wystarczy uruchomić i dodać plik do HOME, aplikacja zadziała.
